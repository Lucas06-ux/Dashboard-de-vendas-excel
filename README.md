# 📊 Dashboard Executivo de Vendas — Level Up Games

## 📌 Sobre o Projeto

Este projeto consiste na criação de um **Dashboard Executivo de Vendas** desenvolvido no **Microsoft Excel Web**, com foco em análise visual de dados e apoio à tomada de decisões.

O dashboard foi criado utilizando **Tabelas Dinâmicas**, **Gráficos Dinâmicos** e **Segmentações de Dados**, permitindo análises rápidas e interativas sobre o desempenho comercial da empresa fictícia **Level Up Games**.

---

# 🎯 Objetivos

O projeto tem como objetivo:

- Transformar dados brutos em informações visuais;
- Facilitar a análise de desempenho de vendas;
- Identificar produtos e categorias mais lucrativas;
- Permitir filtragem dinâmica de dados;
- Desenvolver habilidades em Business Intelligence (BI) utilizando Excel.

---

# 📊 KPIs Apresentados

O dashboard apresenta os seguintes indicadores principais:

| KPI | Descrição |
|---|---|
| 💰 Receita Total | Soma total das vendas realizadas |
| 📈 Lucro Total | Lucro obtido após os custos |
| 📦 Quantidade Vendida | Total de produtos vendidos |
| 🧾 Ticket Médio | Média de receita por venda |

---

# 📈 Gráficos Utilizados

## Receita por Produto
Gráfico de barras horizontais apresentando os produtos com maior receita.

## Receita por Categoria
Comparação de receita entre as categorias de produtos.

## Forma de Pagamento
Visualização das formas de pagamento mais utilizadas pelos clientes.

---

# 🎛️ Filtros Interativos

Foram utilizadas **Segmentações de Dados (Slicers)** para permitir filtragem dinâmica do dashboard.

### Filtros disponíveis:
- Categoria
- Forma de Pagamento

Todos os gráficos e KPIs são atualizados automaticamente ao aplicar filtros.

---

# 🗂️ Estrutura da Base de Dados

A base de dados utilizada contém aproximadamente **100 registros fictícios**, simulando vendas reais de uma loja de games e eletrônicos.

## Campos presentes:

| Campo | Descrição |
|---|---|
| Produto | Nome do produto vendido |
| Categoria | Categoria do produto |
| Quantidade | Quantidade vendida |
| Receita | Valor total da venda |
| Lucro | Lucro obtido |
| Forma_Pagamento | Método de pagamento utilizado |
| Cliente | Nome do cliente |
| Região | Região da venda |

# 🧠 Estrutura do Dashboard

O dashboard foi organizado em três áreas principais para facilitar a visualização e navegação das informações.

## Painel lateral
Responsável pelos elementos de navegação e filtragem:

- Logo da empresa
- Segmentações/Filtros interativos

## Área principal
Responsável pela apresentação visual dos dados:

- KPIs
- Gráficos dinâmicos
- Indicadores visuais

## Área auxiliar
Utilizada para funcionamento interno do dashboard:

- Tabelas Dinâmicas ocultas

# ▶️ Como Reproduzir o Projeto

## 1️⃣ Abrir o arquivo Excel

Abra o arquivo `.xlsx` utilizando uma das opções abaixo:

- Microsoft Excel Web
- Microsoft Excel Desktop

---

## 2️⃣ Acessar a aba Dashboard

O dashboard principal está localizado na aba:


- Dashboard
## 3️⃣ Utilizar os filtros

Use as segmentações laterais para filtrar os dados por:

- Categoria
- Forma de Pagamento

Os gráficos e indicadores serão atualizados automaticamente.

## 📌 Observações

Devido às limitações da versão gratuita do Excel Web, algumas funcionalidades avançadas presentes na versão Desktop possuem restrições. Para contornar essas limitações, foram utilizadas tabelas dinâmicas auxiliares e conexões manuais entre segmentações e gráficos.
